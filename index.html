Performance

The source codes are developed in C++17, using the GNU C++ 8.4.0 compiler with the AVX2 instructions on the Ubuntu 18.04 operating system. All the implementations used in the experiments are compiled at the -O3 optimization level. For the instantiation of the XOF, we use SHAKE in XKCP library. We use SHAKE128 for AIMer-I, and SHAKE256 for AIMer-III and AIMer-V. Our experiments are measured in Intel Xeon E5-1650 v3 @ 3.50GHz with 128 GB memory.
